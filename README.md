## Config
- **multiCore:** specifies whether to use multiple cores or not.
- **name:** the name of the blog, appears in the heading and title of all pages.
- **mobileThemeColor:** the color of the URL bar on mobile browsers
- **gitRepository:** a link to a publicly viewable git repository for this blog which will provide a link to suggest edits to articles. If left empty, this link will be disabled.
- **forceHttps:** specifies whether to force HTTPS when accessed via HTTP.
- **enableCompression:** specifies whether to enable gzip compression, disable if it causes havoc with your server (but it's 2019, it really shouldn't...)
- **copyright:** a copyright notice placed on the footer of all pages.
- **navigation.header:** links that will be displayed on the header of all pages.
